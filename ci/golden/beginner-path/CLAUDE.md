# Project Overview

TaskFlow is a Next.js web application (App Router). React 19 frontend.
Starter scaffold generated by /guardians-of-the-claude:create — flesh
out sections as the project grows.

## Build & Run

npm run dev          # starts dev server on :3000
npm run build        # production build
npm start            # serves the production build

## Testing

Not yet configured. Add a test runner (Vitest / Jest / Playwright) and

## Code Style & Conventions

- Follow Next.js App Router conventions (app/ directory, server
  components by default, "use client" only when needed)
- Use TypeScript strict mode when .ts/.tsx files are added
- Keep shared logic in src/lib/; keep route handlers thin

## Important Context

- Starter configuration — no custom rules, hooks, or agents yet
- Run /guardians-of-the-claude:audit periodically to detect drift
- Run /guardians-of-the-claude:secure when adding secrets/env handling
